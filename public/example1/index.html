User Timing

Demonstrates creating a custom UserTiming with the UserTiming API and a specified timeout value.

Code Breakdown

User Timing

The User Timing is created with the custom name supplied in the form field. The integer value is used as part of a timeout. 

                        
const name = timing.value;
performance.mark(`${name}_start`);
sampleIterateAdd();
setTimeout(() =>{
    performance.mark(`${name}_end`);
    performance.measure(name, `${name}_start`, `${name}_end`);
},timerInt);
Operation

This is the action we are performing before we complete our mark. When the User Timing is recorded the function `sampleIterateAdd` can be found in the timeline.

                
const sampleIterateAdd = () => {
    for(let i=0; i < 100; i++) {
        console.log('Number is now', i);
    }
}
